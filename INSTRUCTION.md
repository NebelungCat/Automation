# Инструкция по настройке автоматизации задач на Windows 10

## 📁 Файлы для использования

1. **DailyTasks.ps1** - PowerShell скрипт с задачами
2. **RunDailyTasks.bat** - BAT-файл для запуска скрипта
3. **Этот файл** - Инструкция по настройке

---

## ⚙️ НАСТРОЙКА СКРИПТА

### Шаг 1: Отредактируйте пути в DailyTasks.ps1

Откройте файл `DailyTasks.ps1` в блокноте и измените следующие переменные:

```powershell
$sourceFile = "C:\Users\Nebelung\Documents\Finance.icash"
$destinationFolder = "D:\YandexDisk\Архив\iCash\"
$torrentFolder = "C:\Users\Nebelung\Downloads"  # Укажите вашу папку
```

**Важно:** Измените `$torrentFolder` на актуальный путь к папке, где хранятся .torrent файлы.

---

## 🚀 ЗАПУСК ВРУЧНУЮ

### Первый тестовый запуск:

1. Скопируйте оба файла (`DailyTasks.ps1` и `RunDailyTasks.bat`) в любую папку
2. Кликните правой кнопкой мыши на `RunDailyTasks.bat`
3. Выберите **"Запустить от имени администратора"**
4. Проверьте результат в консоли

---

## ⏰ НАСТРОЙКА АВТОМАТИЧЕСКОГО ЗАПУСКА (Планировщик заданий)

### Шаг 1: Открыть Планировщик заданий

1. Нажмите `Win + R`
2. Введите `taskschd.msc`
3. Нажмите Enter

### Шаг 2: Создать задачу

1. В правой панели выберите **"Создать задачу..."**
2. Заполните поля:
   - **Имя:** `Ежедневная архивация iCash`
   - **Описание:** `Копирование Finance.icash и очистка .torrent файлов`
   - ✅ **"Выполнять для всех пользователей"**
   - ✅ **"Выполнять с наивысшими правами"**

### Шаг 3: Настройка триггера (когда выполнять)

1. Перейдите на вкладку **"Триггеры"**
2. Нажмите **"Создать..."**
3. Настройте:
   - **Начать задачу:** По расписанию
   - **Однократно:** Ежедневно
   - **Время:** Например, 23:00 (перед окончанием рабочего дня)
   - ✅ **"Включено"**
4. Нажмите OK

### Шаг 4: Настройка действия (что выполнять)

1. Перейдите на вкладку **"Действия"**
2. Нажмите **"Создать..."**
3. Заполните:
   - **Действие:** Запуск программы
   - **Программа или сценарий:** `PowerShell.exe`
   - **Добавить аргументы:** 
     ```
     -ExecutionPolicy Bypass -File "C:\Путь\К\Папке\DailyTasks.ps1"
     ```
   - **Начать в:** `C:\Путь\К\Папке\`
4. Нажмите OK

### Шаг 5: Дополнительные настройки

1. Перейдите на вкладку **"Условия"**
   - ❌ Снимите галочку **"Запускать только при питании от электросети"** (для ноутбуков)
   - ❌ Снимите галочку **"Запускать только при наличии сети"** (если не критично)

2. Перейдите на вкладку **"Параметры"**
   - ✅ **"Запускать независимо от входа в систему"**
   - ✅ **"Не перерисовывать"**
   - **"При сбое задачи перезапустить:"** каждые 5 минут, до 3 раз

### Шаг 6: Сохранение

1. Нажмите OK
2. Введите пароль вашей учетной записи Windows (если потребуется)

## ✅ ПРОВЕРКА РАБОТЫ

### После настройки:

1. В Планировщике найдите вашу задачу в списке
2. Кликните правой кнопкой → **"Выполнить"**
3. Проверьте:
   - Файл появился в `D:\YandexDisk\Архив\iCash\` с датой в имени
   - .torrent файлы удалены из указанной папки

### Просмотр журнала выполнения:

1. В Планировщике выберите вашу задачу
2. Внизу будет вкладка **"Журнал событий"**
3. Или посмотрите историю выполнения в свойствах задачи

## 🔧 ДОПОЛНИТЕЛЬНЫЕ ВОЗМОЖНОСТИ

### Добавить логирование в файл

Измените первую строку в `DailyTasks.ps1`:

```powershell
Start-Transcript -Path "C:\Logs\DailyTasks_$(Get-Date -Format 'yyyy-MM-dd').log" -Append
```

И в конец добавьте:

```powershell
Stop-Transcript
```

### Добавить отправку уведомления

После выполнения задач можно добавить отправку email или push-уведомления.

### Очистка старых архивов

Добавьте в скрипт удаление файлов старше 30 дней:

```powershell
# Удаление файлов старше 30 дней
$daysToKeep = 30
$cutoffDate = (Get-Date).AddDays(-$daysToKeep)
Get-ChildItem -Path $destinationFolder -File | 
    Where-Object { $_.CreationTime -lt $cutoffDate } | 
    Remove-Item -Force
```

## ⚠️ ВАЖНЫЕ ЗАМЕЧАНИЯ

1. **Яндекс.Диск должен быть установлен** и синхронизирован
2. **Проверьте свободное место** на диске D:
3. **Исходный файл остается на месте** (скрипт копирует, а не перемещает)
4. **Для работы по расписанию** компьютер должен быть включен в заданное время
5. **Первый запуск сделайте вручную** для проверки корректности путей

## 🆘 ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

| Проблема | Решение |
|----------|---------|
| Скрипт не выполняется | Запустите от имени администратора |
| Файл не копируется | Проверьте пути и наличие Яндекс.Диска |
| Задача не запускается автоматически | Проверьте права и пароль пользователя |
| .torrent файлы не удаляются | Проверьте путь к папке и права доступа |
