# Автоматизация ежедневных задач

Этот проект содержит скрипты для автоматизации рутинных задач на Windows 10:
1. **Резервное копирование файла iCash** с паролем в Яндекс.Диск.
2. **Автоматическое удаление .torrent файлов** из папки загрузок.

## Требования

- Windows 10 или выше
- PowerShell (встроен в систему)
- Установленный **WinRAR** по пути: `C:\Program Files\WinRAR\WinRAR.exe`
- Папка Яндекс.Диска: `d:\YandexDisk\Архив\iCash\`

## Что делает скрипт

### 1. Резервное копирование Finance.icash
- Копирует файл `C:\Users\Nebelung\Documents\Finance.icash`
- Создаёт архив с именем `Finance_ГГГГММДД_ЧЧММСС.rar`
- Защищает архив паролем: `AlexCas@12`
- Сохраняет архив в `d:\YandexDisk\Архив\iCash\`
- **Важно:** Внутри архива содержится только сам файл, без структуры папок.

### 2. Очистка от .torrent файлов
- Сканирует папку `D:\Downloads`
- Удаляет все файлы с расширением `.torrent`
- Ведёт лог удалённых файлов

## Настройка

Если ваши пути отличаются от стандартных, отредактируйте файл `DailyTasks.ps1` в начале скрипта:

```powershell
$sourceFile = "C:\Путь\К\Вашему\Finance.icash"
$destFolder = "D:\Путь\К\Яндекс.Диску\Архив\iCash"
$TorrentPath = "D:\Путь\К\Папке\Загрузок"
$winRarPath = "C:\Путь\К\WinRAR.exe"
$password = "Ваш_Пароль"
```

## Установка автоматического запуска

Чтобы скрипт выполнялся ежедневно автоматически:

1. Нажмите `Win + R`, введите `taskschd.msc` и нажмите Enter.
2. В правой панели выберите **«Создать задачу...»**.
3. На вкладке **«Общие»**:
   - Название: `Ежедневное резервное копирование`
   - Поставьте галочку **«Выполнять с наивысшими правами»**
4. На вкладке **«Триггеры»**:
   - Нажмите **«Создать...»**
   - Выберите **«Ежедневно»**
   - Укажите удобное время (например, 01:00)
5. На вкладке **«Действия»**:
   - Нажмите **«Создать...»**
   - Действие: **«Запуск программы»**
   - Программа: `powershell.exe`
   - Аргументы: `-ExecutionPolicy Bypass -File "D:\Downloads\Automation-main\DailyTasks.ps1"`
   - Начать в: `D:\Downloads\Automation-main\`
6. Нажмите **ОК** и введите пароль администратора при запросе.

## Логирование

Все действия записываются в файл `tasks_log.txt` в папке проекта. Формат записи:
```
[2026-03-19 01:07:02] Архив успешно создан: Finance_20260319_010702.rar
[2026-03-19 01:07:02] Удален файл: example.torrent
```

## Безопасность

- Пароль от архивов хранится в скрипте в открытом виде. Убедитесь, что файл скрипта защищён от несанкционированного доступа.
- Скрипт требует прав администратора для удаления некоторых файлов и работы с планировщиком.

## Решение проблем

### WinRAR не найден
Убедитесь, что WinRAR установлен по пути `C:\Program Files\WinRAR\WinRAR.exe`. Если путь другой, измените переменную `$winRarPath` в скрипте.

### Файлы .torrent не удаляются
- Закройте торрент-клиент перед запуском скрипта.
- Запустите PowerShell от имени администратора.

### Ошибки кодировки
Файл скрипта сохранён в кодировке **UTF-8 с BOM**. Не редактируйте его в программах, меняющих кодировку (например, старый Блокнот без сохранения BOM).
